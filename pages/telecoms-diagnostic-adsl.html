CONTEXTE

Un vrai problème opérateur

Chez Orange, des équipes entières sont dédiées à détecter les perturbateurs électromagnétiques (moteurs d'ascenseur, alimentations défectueuses…) qui dégradent les lignes ADSL des abonnés et font chuter les débits. Ce projet répond à une vraie problématique opérateur : construire une application web complète permettant à un technicien de se connecter à n'importe quel DSLAM, d'analyser les paramètres de la ligne, et de comparer son état actuel avec son état normal stocké en base de données.

Architecture globale et modulation OFDM/DMT (Cliquez pour agrandir)

OBJECTIFS

Ce qu'on devait réaliser

Côté Télécoms
  • 1 Comprendre la modulation OFDM/DMT utilisée en ADSL/VDSL2
  • 2 Analyser les 4 paramètres de ligne : QLN, SNR, HLOG et BitAlloc
  • 3 Étudier l'impact de perturbateurs électromagnétiques sur une ligne de 2 km réelle
Côté Informatique
  • 4 Concevoir une API REST asynchrone avec FastAPI et Python
  • 5 Développer le front-end en HTML/CSS/JavaScript avec Chart.js pour l'affichage des courbes
  • 6 Implémenter une base de données NoSQL MongoDB pour stocker les profils de lignes
  • 7 Automatiser la connexion SSH aux DSLAM avec Paramiko/asyncssh
ARCHITECTURE

Architecture 3-tiers

Couche Technologie Rôle
Front-end Nginx + HTML/JS/Chart.js Interface web, affichage des courbes, requêtes HTTP
Back-end Python FastAPI (async) API REST, proxy vers DSLAM et MongoDB
Base de données MongoDB (NoSQL) Stockage des profils de lignes en BSON
Connexion DSLAM Paramiko / asyncssh Connexion SSH, commandes CLI, parsing

Architecture serveur — back-end Python / FastAPI déployé sur Linux (Cliquez pour agrandir)

FONCTIONNEMENT

Parcours utilisateur

Étape 01

Sélection du DSLAM

Menu déroulant alimenté dynamiquement via GET /api/dslams. Le technicien choisit l'équipement à interroger.

Étape 02

Choix des paramètres

Sélection de la ligne, du paramètre (QLN / SNR / HLOG / BitAlloc / Tous) et du mode : analyse, comparaison ou enregistrement.

Étape 03

Affichage des résultats

Graphique Chart.js dans le navigateur avec support multi-axes pour afficher les 4 paramètres simultanément.
